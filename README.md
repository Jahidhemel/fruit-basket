# Fruit Burst — Tailwind CSS Landing Page

A fully responsive, modern e-commerce landing page for a fictional organic fruit shop, built from scratch with **Tailwind CSS** and vanilla JavaScript.

🌐 **Live Demo:** [https://jahidhemel.github.io/fruit-basket/](https://jahidhemel.github.io/fruit-basket/)

---

## ✨ Features

- 🎨 Built entirely with **Tailwind CSS** — utility-first styling, zero handwritten CSS
- 📱 Mobile-first responsive design (375px → 4K)
- ☰ Mobile hamburger menu with smooth toggle
- ⏱ **Real working countdown timer** — updates every second
- 🛒 Hover animations on product cards (lift + scale)
- 🎯 Bento-grid "Why Choose Us" section
- 💬 Custom toast notifications + modal dialog (no libraries)
- 📩 Newsletter signup with email validation
- ⌨️ Smooth in-page scroll between sections
- 🌐 Working footer social links (LinkedIn, Facebook, Email)
- 📅 Auto-updating copyright year
- 🧭 Sticky translucent header with backdrop blur
- ⚡ Single HTML file + single JS file — no build step needed

---

## 🛠 Tech Stack

| Tool | Purpose |
|------|---------|
| **Tailwind CSS** (Play CDN) | All styling and responsive layout |
| **HTML5** | Semantic markup |
| **JavaScript** (vanilla) | Countdown, modal, toast, mobile nav |
| **Font Awesome 4.7** | Iconography |
| **Google Fonts (Poppins)** | Typography |

## 🎨 Design Notes

- **Color palette:** Tailwind's `green-500` for the brand, `orange-400` as accent, soft pastel section backgrounds
- **Typography:** Poppins, weights 400–800
- **Spacing rhythm:** Tailwind's 4px scale, generous breathing room between sections
- **Component patterns:** rounded corners (`rounded-3xl`), soft shadows on hover, gradient backgrounds for CTA sections
